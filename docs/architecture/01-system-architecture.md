## 7. Disaster Recovery

### 7.1 Backup стратегия

| Тип | Частота | Хранение | Метод |
|-----|---------|----------|-------|
| PostgreSQL full | Ежедневно | 30 дней | pg_dump + S3 |
| PostgreSQL WAL | Непрерывно | 7 дней | pg_basebackup |
| Redis RDB | Каждые 15 мин | 24 часа | SAVE + S3 |
| Файлы (S3) | Непрерывно | 90 дней | Versioning |

### 7.2 RTO/RPO

| Компонент | RTO (Recovery Time) | RPO (Recovery Point) |
|-----------|---------------------|----------------------|
| PostgreSQL | < 1 часа | < 15 минут |
| Redis | < 30 минут | < 15 минут |
| API | < 15 минут | 0 (stateless) |
| Frontend | < 5 минут | 0 (CDN) |
